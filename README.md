
# learnandhelp
Learn and Help:  Learn programming and help different causes.

Steps to Deploy:

1. Download the Repo
2. Extract the contents to xampp/htdocs/learnandhelp_2026
3. Start XAMPP
4. Create a database called "learnandhelp_db"
5. impor the SQL file from xampp/htdocs/learnandhelp_2026/sql/learnandhelp_db.sql into learnandhelp_db
6. You can now launch teh app http://localhost/learnandhelp_2026


If you want to login as admin:
7. You can also create an account as a regular user
8. Go to your phpmyadmin / leaarnandhelp_db / users table.
9. Find the account you craeted.
10. Change the Role from user to admin to get the admin access.
